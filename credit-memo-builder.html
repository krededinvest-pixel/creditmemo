How to Host and Share This Tool

Option 1 - GitHub Pages (recommended for sharing a link):
1. Create a free account at github.com
2. Create a new repository and upload this HTML file, renaming it to index.html
3. Go to Settings > Pages > set Source to "main" branch > Save
4. Your link will be: https://yourusername.github.io/repositoryname

Option 2 - Share the file directly:
Email or share the HTML file. Recipients open it in any modern browser (Chrome, Edge, Firefox, Safari).

API Key: Each user must enter their own Anthropic API key. Keys are available at console.anthropic.com. The key is stored only in the browser and never sent anywhere except directly to Anthropic's API.
