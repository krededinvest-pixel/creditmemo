How to save, host, and share this tool

Save locally

  1. Right-click this page in your browser and choose Save As
  2. Save as credit-memo-builder.html (Webpage, HTML Only)
  3. Open the saved file directly in any modern browser - no server needed

Host on GitHub Pages

  1. Create a free account at github.com
  2. Create a new public repository (e.g. credit-memo-builder)
  3. Upload credit-memo-builder.html and rename it to index.html
  4. Go to Settings → Pages → set Source to main branch, root folder
  5. Your tool will be live at https://yourusername.github.io/credit-memo-builder

Share with colleagues

  1. Once hosted on GitHub Pages, share the URL directly
  2. Or email the .html file - recipients can open it locally without installing anything
  3. All data is saved automatically in the browser - each user has their own private copy
