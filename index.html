How to Host and Share This Tool

Option 1 - GitHub Pages:
1. Create a free account at github.com
2. Create a new repository and upload this file renamed to index.html
3. Go to Settings > Pages > set Source to main branch > Save
4. Share the link: https://yourusername.github.io/repositoryname

Option 2 - Share the file directly: Email or share the HTML file. Recipients open it in Chrome, Edge, Firefox, or Safari.

API Key: Each user needs their own Anthropic API key from console.anthropic.com. The key is stored only in their browser.
